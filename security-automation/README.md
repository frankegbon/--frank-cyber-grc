# Security Automation Scripts

## Overview
A collection of lightweight automation scripts designed to reduce manual effort, improve audit readiness, and enforce secure configurations across cloud and on-prem environments.

## Key Features
- Automated evidence collection
- Baseline configuration checks
- Log integrity verification
- Secure provisioning workflows

## Why This Matters
Security automation reduces operational overhead, improves consistency, and strengthens compliance posture for SOC 2, ISO 27001, and internal audits.

## Scripts Included
- windows-baseline-check.ps1
- linux-log-integrity.sh
- azure-secure-provisioning.ps1
