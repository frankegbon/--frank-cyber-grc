# GRC Toolkit

## Overview
A practical set of templates, scoring models, and workflows for SOC 2, ISO 27001, vendor risk management, and audit readiness.

## Key Features
- Risk register templates
- Vendor risk scoring engine
- Audit readiness checklists
- Control mapping samples

## Why This Matters
GRC teams need repeatable, scalable processes. This toolkit provides structure and clarity for organizations preparing for audits or maturing their governance programs.
