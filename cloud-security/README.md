# Cloud Security Reference Architecture (Azure)

## Overview
A set of secure-by-default Azure architecture patterns, identity governance models, and IaC templates for enterprise cloud environments.

## Key Features
- Azure landing zone blueprint
- Identity governance patterns
- Secure IaC templates (Terraform/Bicep)
- Network segmentation examples

## Why This Matters
Modern cloud environments require strong identity, network, and governance foundations. These patterns help teams deploy secure, scalable Azure environments.
